Android (JAVA)

Android Unit Test 测试框架

  • Mockito + PowerMock
  • JMockit : 它相当于一个非插件化的 PowerMock，集成和使用的方式都非常简单
  • Robolectric: 重写了许多 Android SDK 里的类，使得在 JVM 上进行 Android 测试成为了可能。
  • Espresso: 由Google开源的一款最新的Android自动化测试框架

TestDroid Recorder, 将操作记录成文本的工具

Android UI测试框架

  • Monkey: 一种产生随机事件的测试方法，使用adb shell monkey命令进行测试，是测试程序稳定的最佳方案。
  • Monkeyrunner: UI 测试框架 - 适用于 简单的点按滑操作，测试代码使用Python
