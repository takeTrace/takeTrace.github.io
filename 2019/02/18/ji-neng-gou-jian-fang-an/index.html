常用快捷键

  • 代码对齐: 打开控制板 -> 输入reindent Lines
  • 指定当前文件解析的格式: 打开控制面板 -> 直接输入想要的格式, 找不到的话, 输入完整的命令 key: set syntax: -> 找到对应的格式

文件跳转

使用 Goto Anything, 可以实现跳转到任意的地方

  • 跳转都某个文件, 输入文件名中的某几个字就行, 会模糊匹配
  • 路径后跟文件名: 就能更精确
  • 文件名后加: + 行号, 能跳到对应行
  • 文件名后+@+函数: 跳到对应函数
    • #: 查找字符串
