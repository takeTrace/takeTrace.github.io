值得付费的 App / 服务

发表于 2019-04-17 | 分类于 Tech |

现在的生活, 已经离不开现代的网络和技术了. 不是几乎, 是完全离不开.

无论直接间接, 即使不用任何电子产品, 只要生活, 多多少少都间接受到影响.

我也算是个工具党吧.

长期以来, 为了提高效率, 都积累了很多软件, 开源的, 和谐的等等. 现在越来越多的服务已经开始云化了. 而且这种趋势能极大的提高了便利程度. 而且对于和谐, 更多的不能使用服务了.

最近一段时间, 我也开始为提高自己的效率的便利付费. 这里简单讲讲几个我觉得值得去付费的服务和软件

EverNote

免费的有60M, 下一级就是10G, 跳的很大是吧. 但是60M 每月的量真的…. 截取些网页就已经不够了.

本来想省着用, 一直用免费的每月60M 就好了, 但是, 他的连接一切的实际效果, 真的是很方便啊. 直接打败其他收录软件, 没有之一, 相信我, 没有之一. 它建立出的生态和工具, 几乎在任何地方都可以将你要的东西都放进去.

以下是你几乎能获取资料的场景:

  • 微信: 这个已经是实际上的互联网基建型应用. 拥有大量的信息, 很多人的很多信息, 都是从这里获取的. 通过印象笔记的公众号我的印象笔记(国际版是MyEvernote), 将需要收藏的文章转发过去. 就能自动保存到笔记中, 并加上微信/wechat的标签.
  • 微博: 一样, 通过@公众号, 就会自动转存. 并加上微博/weibo 标签 (注: 国际版没有公众号. 但是国际版微博有分享到Evernote 的选项<但是分享出去的文本效果并不是太好, 有些原po 会消失>)
  • 浏览器: 这是任何 web 几乎的必经之地了. 印象笔记有浏览器插件, 可以方便的保存正在浏览的网页. 或者简化版, 或者截图. 这点几乎能满足你80%的信息收藏需求了.
  • 电脑桌面: 印象笔记也有桌面客户端. 而且有个功能就是可以随时唤出一条笔记, 可以截图, 录音什么的.
  • 其他应用: 大部分信息类软件或者服务, 都整合了 Evernote 的链接服务. 比如 Inoreader, MarginNote 等.
  • 万能的邮件方式: 就算没有应用集成, 各软件(起码在 Mac 平台下)天然的具有分享的功能都支持通过邮件分享. 直接发到印象笔记给配的私人邮箱就好了. (注, 手机上的发送邮件在有些 app 上支持不好. 比如邮件内容只会有一些摘要和标题. 但是可以通过先在 Safari 打开, 在发送邮件解决)

以上这些, 只是收集.
下面这里, 还有更强大的.

  • 标签管理
    收集来的资料, 我们不仅可以分笔记本管理(不推荐太多笔记本, 下面讲到), 还可以通过标签管理.
    通过给笔记打上不同的多个标签, 可以对笔记进行多维度的分类. 以后我们可以按照标签 层层搜索出想要的分类, 重要的是, 标签在输入过程中会自动过滤, 即如果输入了标签A, 剩下的待选的标签只会显示和A同时出现的其他标签, 帮助了记忆的链接.

所以, 笔记本可以做大的分类, 其他维度交给标签来管理.

  • 全文搜索:
    有时候我们不太记得一些东西, 但是只记得一些片段, 一些只言片语, 你去网上搜, 可能不太好搜到了, 历史纪录又只显示标题. 这时如果你已经习惯于把觉得可以归档的东西放到印象笔记, 那么他的全文搜索, 你一定会依赖上. 可以很快的找到你当时看的那些东西.

  • 图片/pdf 搜索
    没错, 即是图片离出现的字, 也会给您搜到并框出来…真是很惊喜了. PDF 更不用说了, 看过的电子书扔进去, 以后搜起来就方便了, 即使是扫描档也无所谓! 爽爆

MarginNote

关于知识获取, 除了信息更新在网上无疑是最方便最快捷最新鲜的, 但是最有积累的, 一定是编辑成册的书.

实体书无疑是最让人有沉浸感的, 但是对于知识的组织回顾效率, 还有携带型便捷性上, 如今已经远远不如电子书.

如果只是一般的使用些 PDF 软件来阅读. 大概不会感觉到我说的那些, 顶多是能便捷的看到自己的标注罢了. 也就局限在了这个软件对于 PDF 的优化和体验方面怎么怎么样.

但是, 如果有这么一款软件, 是专门为电子书的知识获取和整理回顾深入学习来设计开发的, 那你就会知道我想表达的是什么了.

MarginNote 就是这么款软件之一. 对, 是之一, 还有一些其他的. 但是, 一是它现在在 Setapp 中有, 我能直接正版使用, 而是他的学习设计方式正是我想要的.

关于使用, 上网搜一下就有很多帖子了. 去浏览一篇, 就知道它很不错.

简单讲下.

他在 PDF 胖会有一个边距, 专门展示你做的笔记. 可以单独看笔记列表, 做的比较可以贴图, 涂鸦等等方便你记忆索引的东西

通过看大纲, 你觉得适合继续深入下去, 可以进行学习模式, 这是一个更强大完善的学习管理模式. 可以对你做的注释自动生成脑图

而且当你读完一本书, 可以将这全部的笔记带书本导出到印象笔记!
这样这些东西, 就成为你外部脑的一部分了. 除了在MarginNote 可以复习搜索外, 当你有记忆片段想还原时, 在印象笔记里搜索, 也不会错过你在这里学习时的所做所知所看.
